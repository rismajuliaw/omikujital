スタジオの物語

We translate shrine rituals into daily desktop pauses.

Omikujital is a two-person studio in Kamakura building small, considerate software. We blend research, calligraphy, and interaction design so idioms feel tactile even through glass.

哲学

Design code

Every interface references a physical ritual: paper folds, incense smoke, lacquer trays. That anchor keeps the product from drifting into empty UI flourishes.

言葉

Word sourcing

We log idioms from Edo-era collections, community submissions, and elders in Shizuoka. Each entry earns romaji, context, and a friendly paraphrase.

体験

Experience

The deck lives on phones, desktops, and a physical zine. Tap, drag, or shuffle—the same idiom is mirrored everywhere, but the pacing changes with the medium.
